	INDICACIONES 

	Ejecutar en terminal con el comando "javac Prueba.java" seguido de "java Prueba.java"

		El programa pide 3 datos para trabajar
		1.- Largo y alto del laberinto, ambos datos deben de ser válidos, es decir mayores a cero
		2.- Coordenada de partida 
		3.- Coordenada de fin

			** Las dos coordenadas deben de ser válidas, es decir, deben de contener valores dentro del largo y alto del laberinto 

		Una vez que se han introducido los valores, el programa imprime el tablero inicial, seguido del tablero con un laberinto creado así como con el laberinto resuelto con las coordenadas proporcionadas

		En caso de ingresar un dato no válido el programa lo señala en pantalla.
		nota: Cada casilla utiliza 3x3 caracteres para imprimirse, se recomienda tenerlo en cuenta al seleccionar las dimensiones del tablero
